Developer Integration
Embed this browser in your own app with two lines of HTML

The <scsp-community> web component renders a self-contained capability browser using Shadow DOM. Drop it anywhere in your admin UI.

<script src="https://scsp.dev/sdk/widget.js"></script> <scsp-community registry="https://raw.githubusercontent.com/scsp-community/registry/main/registry/index.json" theme="auto" max-height="560px" ></scsp-community>

Attributes: registry (your registry URL), theme (light/dark/auto), max-height (CSS value). See README for full docs.
